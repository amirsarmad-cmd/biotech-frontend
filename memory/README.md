# Memory Archive

This directory is the durable memory for the Biotech Screener project.

Use it to record:

- Sessions: what happened during a work block.
- Decisions: why the project chose one direction over another.
- Designs: UI, product, data, or architecture sketches.
- Handoffs: what the next agent should read and do.

Start with:

- `CURRENT_STATE.md`
- `INDEX.md`
- `DECISIONS.md`
- `handoffs/NEXT_AGENT.md`

Use `npm run memory:status` to inspect the archive and `npm run memory:index` after edits.
